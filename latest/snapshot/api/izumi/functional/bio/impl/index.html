  9.  sealed trait MiniBIO[+E, +A] extends AnyRef

    A lightweight dependency-less implementation of the BIO interface, analogous in purpose with cats.effect.SyncIO

    A lightweight dependency-less implementation of the BIO interface, analogous in purpose with cats.effect.SyncIO

    Sync-only, not interruptible, not async. For internal use. Prefer ZIO or cats-bio in production.

    This is safe to run in a synchronous environment, Use MiniBIO.autoRun to gain access to components polymorphic over BIO in a synchronous environment.

    final class MyBIOClock[F[+_, +_]: BIO]() {
  val nanoTime: F[Nothing, Long] = BIO(System.nanoTime())
}

import MiniBIO.autoRun._

val time: Long = new MyBIOClock().nanoTime
println(time)
